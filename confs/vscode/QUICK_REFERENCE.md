# VS Code Vim - Quick Reference

Quick reference for your Neovim-to-VS Code key mappings.

## Leader Key: `<Space>`

## üìù Editing

| Key              | Action                   |
| ---------------- | ------------------------ |
| `<C-s>`          | Save all                 |
| `<S-u>`          | Redo                     |
| `ff`             | Format document          |
| `<C-c>` (visual) | Copy to clipboard        |
| `<A-j>`          | Move line/selection down |
| `<A-k>`          | Move line/selection up   |

## üìÅ Files & Navigation

| Key          | Action                  |
| ------------ | ----------------------- |
| `<C-p>`      | Open file (fuzzy)       |
| `<C-f>`      | Find in current file    |
| `<leader>ee` | Toggle file explorer    |
| `<leader>ef` | Reveal file in explorer |
| `<leader>fw` | Search in workspace     |

## üîÄ Git

| Key           | Action       |
| ------------- | ------------ |
| `<leader>gf`  | Git files    |
| `<leader>gc`  | Git commits  |
| `<leader>gst` | Git status   |
| `<leader>gbr` | Git branches |

## üîß LSP/Code

| Key          | Action                |
| ------------ | --------------------- |
| `gd`         | Go to definition      |
| `gD`         | Go to declaration     |
| `gR`         | Show references       |
| `gi`         | Go to implementation  |
| `gt`         | Go to type definition |
| `gh`         | Show hover/docs       |
| `<leader>ca` | Code actions          |
| `<leader>rn` | Rename symbol         |
| `<leader>D`  | Show problems         |
| `<leader>er` | Show line diagnostics |
| `<leader>io` | Organize imports      |

## ü™ü Splits

| Key     | Action         |
| ------- | -------------- |
| `<C-h>` | Navigate left  |
| `<C-l>` | Navigate right |
| `<C-k>` | Navigate up    |
| `<C-j>` | Navigate down  |

## üí° Tips

- All keybindings match your Neovim setup
- Relative line numbers are enabled
- System clipboard is automatically synced
- Press `Ctrl+K Ctrl+S` to view all keybindings
